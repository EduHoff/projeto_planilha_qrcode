# projeto sem um nome certo ainda

## Instalação

Pré requisitos:
- Git
- Docker

Linux, MacOS ou Unix-like:
```bash
git clone https://github.com/EduHoff/projeto_planilha_qrcode.git
cd projeto_planilha_qrcode
chmod +x start.sh
./start.sh --build
```

Windows:
```cmd
git clone https://github.com/EduHoff/projeto_planilha_qrcode.git
cd projeto_planilha_qrcode
start.bat --build
```

## Iniciar

```
docker compose up
```

## Encerrar

```
docker compose down
```
